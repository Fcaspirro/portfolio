{Conhecimentos}

HTML

HTML é denominado como uma linguagem de marcação, é utilizada na estruturação de elementos básicos de páginas Web.

CSS

CSS é Utilizado para estilizar elementos escritos em uma linguagem de marcação, acrescentando assim uma estética ao conteúdo.

JAVASCRIPT

JavaScript permite a implementação de itens complexos em páginas web, possibilitando a interação e animação dos componentes exibidos.

TYPESCRIPT

O Typescript permite definir a tipagem estática, parâmetros e retorno de funções..

TAILWIND CSS

Uma estrutura CSS de utilidade que podem ser compostas para construir qualquer design, diretamente em sua marcação.

REACT.JS

O React é a biblioteca mais popular do JavaScript e é usada para construir interface de usuário (UI). Essa stack oferece uma resposta excelente para o usuário adicionar comandos usando um novo método de renderizar sites.

REACT NATIVE

O React Native também é uma biblioteca Javascript, entretanto é utilizada para desenvolvimento de aplicativos nos sistemas Android e iOS de forma nativa.

Node.js

Node.js é um runtime de JavaScript que leva a renderização e processamento do código JavaScript para o lado do servidor, desvinculando-o totalmente do browser, possibilitando que você desenvolva aplicações de rede rápidas e estáveis
